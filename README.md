## 🚀 How to Deploy on Vercel (Free)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** icon → **New repository**
2. Name it: `padalingam-weds-rajarajeshwari`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload your files
1. Click **uploading an existing file**
2. Drag and drop ALL files from this folder:
   - `index.html`
   - `style.css`
   - `script.js`
   - `vercel.json`
3. Also create an `images` folder and upload your photos
4. Click **Commit changes**

### Step 4 — Deploy on Vercel
1. Go to https://vercel.com and sign up with your GitHub account (free)
2. Click **Add New Project**
3. Select your `padalingam-weds-rajarajeshwari` repository
4. Click **Deploy** — no settings needed!
5. In ~30 seconds your site is live! 🎉

### Step 5 — Custom URL (optional)
Vercel gives you a free URL like:
`padalingam-weds-rajarajeshwari.vercel.app`

You can also set a custom domain if you have one.

## 📸 Adding Photos Later

1. Go to your GitHub repository
2. Click on the `images` folder
3. Click **Add file → Upload files**
4. Upload your photos with these exact names:
   - `couple.jpg` — Main couple photo
   - `gallery1.jpg` to `gallery5.jpg` — Gallery photos
5. Commit — your live site updates automatically!
